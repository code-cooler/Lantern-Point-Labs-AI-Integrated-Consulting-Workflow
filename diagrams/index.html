About the Consulting Workflow Tool

This interactive tool provides a visual map of the Lantern Point Labs AI-enhanced consulting workflow. It is designed to help consultants:

  • Navigate through the consulting process from project intake to completion
  • Access detailed guidance for each phase
  • Leverage AI capabilities at key decision points
  • Ensure consistency in approach across different client engagements

The workflow is structured as a main diagram with detailed subtrees for each major phase, allowing you to drill down into specific processes as needed.

How to Use This Tool

Follow these steps to effectively use the consulting workflow tool:

  1. Start with the Main Diagram: Use this to get an overview of the entire consulting process.
  2. Click on Phase Boxes: Click any main phase box (e.g., "Project Intake & Scoping") to view its detailed subtree.
  3. Explore AI Integration Points: AI nodes (purple with dashed borders) include tooltips with recommended prompts.
  4. Navigate Between Diagrams: Use the "Back to Main Flow" links to return to the main diagram.
  5. Use Zoom Controls: Adjust the view using the zoom buttons if needed.

For more detailed information, refer to the Decision Tree Guide and AI Consulting Guide.
